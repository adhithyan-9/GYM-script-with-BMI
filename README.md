# GYM-script-with-BMI
This Python Tkinter application helps beginners in the gym by generating a personalized fitness plan. Based on weight, height, and goal, it creates BMI analysis, workout routines, diet plans, and daily water intake. Simple, interactive, and beginner-friendly.
